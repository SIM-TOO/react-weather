# react-weather
1. 날씨 앱
- 서울 외 3곳의 날씨 정보를 알 수 있습니다.
- 현재 기온, 습도, 기압, 풍속을 표시합니다.
- 다른 날짜의 날씨를 표시를 합니다.

2. API KEY 발급 필요
- openweather 사이트에서 API 발급
- 플랜에 따라 기능이 작동하지 않을 수 있습니다.
- \utils\openWeather.ts API 직접 입력 및 .env에 입력

3. 사용한 라이브러리
 - react-router-dom
 - react-query
 - styled-components
 - react-select
 - date-fns
 - react-icon
 - openweathermap-ts
 - all-the-cities
